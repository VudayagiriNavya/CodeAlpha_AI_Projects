## 📌 Project Details

### 1️⃣ Chatbot for FAQs

**Features:**

* Preprocesses user queries using NLP (tokenization, cleaning).
* Matches queries with FAQs using **cosine similarity**.
* Provides relevant answers in real-time.
* Optional interactive chat UI.

**Applications:**

* Customer support automation
* Information retrieval systems

---

### 2️⃣ Object Detection & Tracking

**Features:**

* Real-time detection using **YOLOv8**.
* Tracks multiple objects with unique IDs using **SORT/Deep SORT**.
* Works with webcam feed or video files.
* Bounding boxes & labels for detected objects.

**Applications:**

* Surveillance & security systems
* Robotics & autonomous vehicles
* Real-time monitoring & analytics

---

### 3️⃣ Offline Translation Tool with Voice Input & TTS

**Features:**

* Offline speech-to-text for multiple languages (20+).
* Text-to-speech (TTS) reads translations aloud.
* Copy button for easy sharing of translated text.
* Clean UI with customizable backgrounds.

**Applications:**

* Multilingual communication
* Travel & language learning
* Offline accessibility applications

## 🛠 Installation & Setup

1. Clone the repository:

```bash
git clone <your-repo-url>
cd <repo-folder>
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the desired project:

```bash
# Chatbot
streamlit run chatbot_app.py  

# Object Detection & Tracking
python object_detection_app.py  

# Offline Translation Tool
streamlit run translation_tool.py
```
